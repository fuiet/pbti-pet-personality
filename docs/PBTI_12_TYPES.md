# Official PBTI 12-Type System

PBTI keeps four scored behavior dimensions and returns one of 12 official four-letter personality types.

| Code | Name |
| --- | --- |
| AEVG | Explorer |
| AECG | Guardian |
| ASCP | Dreamer |
| IEVP | Maverick |
| IECP | Scholar |
| IEVG | Leader |
| ASEP | Companion |
| ASEG | Healer |
| AEVP | Sunny |
| ASCG | Sentinel |
| AECP | Player |
| IECG | Noble |

Cats and dogs use separate 36-question sets. Both species resolve into the same 12-type system, while result artwork is selected from the species-specific asset directory.
